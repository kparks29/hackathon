    Mixins

    Here are examples of the current mixins

  • Icon Background

    Icon background allows you to use svg's as a background image. You just need to specify it's height, width, and background color in addition to the @include.

    div {
    @include iconBackground('icon_concierge.svg');
    background-color: $homdnaDarkFont;
    width: 50px;
    height: 50px;
    }

  • Hide Select Arrow

    Hide the dropdown arrows for the select box. Only requires @include statement.

    Arrows Visible
    Arrows Gone

    select {
    @include hideArrow();
    }

  • Set Box Shadow

    Sets the box shadow for multiple browsers. You must pass in a value.

    div {
    @include iconBackground(0 0 15px 5px #888888); }

  • Hide Number Arrow

    Hide the arrows for the input type number for multiple browsers. Only requires @include statement.

    Arrows Visible
    Arrows Gone

    input[type="number"] {
    @include hideNumberArrow();
    }

  • Breakpoint

    Breakpoint allows you to pass in a device type and content instead of having a bunch of @media rules cluttering up your css. Simply @include breakpoint(device-type) { add-css: rules-here }. It is as simple as that. Currently we support a generic for all smart phones in portrait for iphone6+ to iphone 4 and we also offer support for landscape and iphone4&5 in portrait. See code example below.

    div {
    float: left;
    width: 300px;
    @include breakpoint(smart-phone) { float: none };
    @include breakpoint(smart-phone-landscape) { float: left };
    @include breakpoint(smart-phone-portrait) { float: none };
    }

    Global Variables

    Common variables used through-out the app.

  • $borderRadius

    Is the common border radius for all elements that require one. Currently 3px border radius. You still need to specify a border.

    div {
    border-radius: $borderRadius;
    border: 3px solid $homdnaDarkFont;
    }

  • $letterSpacing

    Is the common letter-spacing for all text elements that require one. Currently 0.5px letter spacing.

    Without Letter Spacing - With Letter Spacing

    p {
    letter-spacing: $letterSpacing;
    }

  • $mainFont

    Is the common font-family for all text elements that require one. Currently source_sans_proregular font-family.

    Without Main Font - With Main Font

    p {
    font-family: $mainFont;
    }

  • Form Select Icon

    Adding a select arrow requires 3 classes, and id, and a label. Firstly wrap your select class in a div tag with a form-input class. Next give your select an id. Then add a label below the select tags, but still within the div tag. Add a for property to the label and give it the same name as the id of the select tag. Next give the label a class of select-icon and icon-downarrow. This will give you a custom downarrow for the select.

    <form>
    <div class="form-input">
    <select id="name">
    <option value="">Choose An Option</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    </select>
    <label for="name" class="select-icon icon-downarrow"></label>
    </div>
    </form>

    Footer

    Footer for the app. The footer modifies and adds to the existing ionic footer. You can specify the text and the icons you use. Be sure to add the icon class when adding icons. Notice the active class, it makes the background a bit darker and the font white. The tab-item-main is slightly larger than the other two. The footer is responsive for phone sizes iphone4 to iphone6+.
